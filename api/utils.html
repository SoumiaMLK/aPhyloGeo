aphylogeo.utils.fasttree(msaset, boot=1000, nt=True)[source]ÔÉÅ

Create phylogenetic trees from a set of multiple sequence alignments using the FastTree application.

This function creates temporary FASTA files to be used as input for FastTree. Since FastTree outputs .tree files, the function reads them back during execution and removes the output tree files afterward.

Parameters:

  • msaset (dict(str, AlignIO)) ‚Äì A dictionary containing the multiple sequence alignments (MSA).

  • boot (int, optional) ‚Äì The number of bootstrap replicates to generate. Default is 100.

  • nt (bool, optional) ‚Äì Whether the sequences are nucleotides (True) or amino acids (False). Default is True (nucleotide sequences).

Returns:

A dictionary where the key is the window name and the value is a tree in Newick format.

Return type:

dict(str, Tree)
